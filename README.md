# Personal Fint

Personal Fint is a full stack web application developed as part of a DevOps practices workshop.

The project focuses on building a simple personal budget control application while applying a modern development and deployment workflow.

## Tech Stack

* Next.js
* TypeScript
* pnpm
* Prisma ORM
* PostgreSQL
* Docker
* SonarQube

## Purpose

This repository was created for a DevOps workshop, with the goal of practicing version control, containerized development, code quality analysis, and continuous integration principles using a full stack application.

## Project Scope

The application allows basic personal budget management through expenses, categories, monthly budgets, recurring expenses, and dashboard views.

This project is intended for academic and learning purposes.
